O scancel é utilizado para cancelar um job, sendo necessário apenas informar o id do job no comando.

Utilização:

<div class="code">
    user@cluster:~$ scancel &lt;id-job&gt;
</div>

\
Exemplo de uso:

<div class="code">
    user@pantanal:~$ scancel 1234
</div>
